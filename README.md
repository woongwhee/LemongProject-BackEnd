<h2>개요</h2>

<li> 각자 크고 작은 목표가 있지만 매번 실패하지 않았나요?
<li> 목표만 설정하고 실천은 뒷전이지 않나요?
<li> 혼자서만 목표를 세우다보니 감시 해줄 사람이 필요하지 않나요?

 <br/>
    

<p> - 이럴때를 위해 저희가 준비했습니다. <p/>
<p>  - 같이하는 ToDoList! <p/>
<p>  - 함께 모여서 공통된 목표로 나아가는 커뮤니티! </p>

<br/><br/><br/>

<h2>기획의도</h2>

<li> 공통된 목표의 사람들과의 교류
<li> 함께 목표를 이룰 수 있는 챌린지 달성
